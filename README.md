# Tao

Tao is an academic energy and productivity management web application
designed specifically for students.

It helps users monitor their daily energy, emotional state, study
patterns, and academic workload in order to detect burnout early and
improve performance without sacrificing well-being.

---

## The Problem

Students often experience:

-   Academic overload
-   Mental fatigue
-   Lack of focus
-   Decreased performance
-   Early burnout symptoms

Existing productivity tools focus on tasks, and wellness apps focus on
meditation --- but none connect energy, stress, and academic performance
in a unified system.

---

## The Solution

Tao connects:

-   Energy tracking
-   Emotional state monitoring
-   Academic workload
-   Productivity metrics

The system analyzes patterns and provides smart recommendations to help
students maintain balance and optimize performance.

---

## Core Features

-   Daily Check-In (energy, mood, stress, sleep)
-   Study Session Tracking
-   Burnout Risk Indicator
-   Smart Recommendations
-   Goal Tracking Dashboard
-   Personalized Onboarding
-   Authentication System

---

## Tech Stack

-   React
-   Vite
-   Firebase (Authentication & Database)
-   React Router
-   External API Integration

## Installation

Clone the repository:

    git clone https://github.com/SmallCakekoo/Tao

Install dependencies:

    npm install

Run locally:

    npm run dev
